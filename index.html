HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The footer contains copyright information and other information such as page authors, etc.
  • The title element is important for SEO, as well as giving the website tab a name.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A border establishes a boundary layer between the margin and the padding.

CSS

  • CSS allows for style to be added to the HTML file.
  • The box model provides a visual representation of the layout of text and other features in a webpage.
  • Below is an image showing the box model:
  • The margin indicates how much space the web developer wants on the outside of their content.
  • The border allows developers to put a box around the content and style it should they desire to do so.
  • The padding allows a buffer between the content itself and the border surrounding it. If the padding is increased, the entirety of the element is increased in size.

Git

  • To save, follow the outline below:
    1. Save changes locally.
    2. Ensure you're working in the correct working branch.
    3. Enter the following command into Git: "git add -A"
    4. Commit changes to GitHub by entering the following: "git commit -m "add a message describing the changes""
    5. Double-check that the local branch is in sync with the base branch by entering the following: "git pull origin main"
    6. Push changes to remote branch by entering the following command: "git push origin 'working branch title'"
    7. Navigate to the appropriate GitHub repo and click "Compare & pull request"
    8. Add notes to the pull request as necessary to describe the changes made.
    9. Once complete click "Create pull request"

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Functions are sets of insructions that tell the computer how to complete a specific task.
    •             
            
              function sayHello(); {
                console.log("Hello everyone!")
              }
  • The code above shows in brief how a function is created.
    1. First, it is given a name. (in this case 'sayHello')
    2. The name of function is then followed by an open and closed parentheses '()' followed by a semicolon ';'.
    3. The content of the function is then added between curly brackets '{' and '}'.
    4. The computer will then perform the steps in the function when it is called to do so.
    5. In the case of the example 'sayHello' function above the function would log in the console the following: Hello everyone!
